# Intel Metadata

This directory contains reusable definitions of common structures and different manifests for Intel platforms.

## Structure
```
├── cbnt                # Converged Boot Guard and TXT
│   ├── bootpolicy      # BG/CBnT Boot Policy Manifest
│   ├── keymanifest     # BG/CBnT Key Manifest
├── common              # Common elements
│   ├── header_revision # Version detection        
│   ├── examples
│   ├── manifestcodegen # Legacy code generator
│   ├── pretty
│   ├── tracedbinary
│   └── unittest
└── fit                 # Firmware Interface Table
```
